Notre Bureau

Pourquoi Nexus ?

"Nexus" est un mot latin qui signifie "lien", "connexion", "point de rencontre".

Nous avons choisi ce nom car il symbolise parfaitement notre ambition : faire du club un carrefour entre les esprits curieux et le monde de la data. C'est aussi l'endroit où les talents se rencontrent pour ensemble aller plus loin.

Notre Vision

Faire du club un espace collaboratif et un environnement où chaque personne peut trouver sa part de connaissances.

Nos Objectifs

  • Rendre la data science accessible à travers les formations
  • Déclencher une motivation particulière des élèves pour la data à travers les challenges
  • Favoriser la collaboration entre les membres

Programme de Formation

Niveau 1 : Fondations en Data Science

Module 1 : Introduction à Python et à l'Algorithmie

  • Bases de Python : types de données, variables, opérations
  • Structures de contrôle : conditions, boucles
  • Fonctions et modules

Module 2 : Manipulation et Visualisation des Données

  • Introduction aux bibliothèques Pandas et NumPy
  • Nettoyage et transformation des données
  • Visualisation avec Matplotlib et Seaborn
  • Statistiques descriptives (moyenne, médiane, écart-type, correlation)

Module 4 : Gestion et Stockage des Données

  • Introduction aux bases de données SQL
  • Lecture et écriture de fichiers CSV, Excel

Module 5 : Programmation Orientée Objet (POO) en Data Science

  • Classes et objets en Python
  • Organisation du code pour des projets data

Module 6 : Introduction au Machine Learning

  • Supervised vs Unsupervised Learning
  • Algorithmes de base : Régression linéaire, k-means, KNN
  • Utilisation de Scikit-learn pour entraîner un modèle

Projet de fin de Niveau 1 : Analyse Exploratoire de Données

  • Étude des tendances d'un marché (ex : analyse des prix sur Jumia)
  • Visualisation et interprétation des données
Niveau 2 : Machine Learning et Applications Avancées

Module 1 : Programmation Orientée Objet (POO) en Data Science

  • Classes et objets en Python
  • Organisation du code pour des projets data

Module 2 : Introduction au Machine Learning

  • Supervised vs Unsupervised Learning
  • Algorithmes de base : Régression linéaire, k-means, KNN
  • Utilisation de Scikit-learn pour entraîner un modèle

Module 3 : Feature Engineering et Optimisation des Modèles

  • Sélection et transformation des variables
  • Validation des modèles et métriques d'évaluation
  • Optimisation des hyperparamètres

Module 4 : Web Scraping et Collecte de Données

  • Extraction de données avec BeautifulSoup et Scrapy
  • Automatisation de la collecte de données
  • Stockage des données récupérées

Module 5 : Déploiement d'un Modèle de Machine Learning

  • Introduction à Streamlit pour créer une interface interactive
  • Déploiement d'un modèle sur le web
  • Présentation et restitution des résultats

Projet de fin de Niveau 2 : Modélisation et Prédiction

  • Développement d'un modèle prédictif (ex : prédiction des prix d'un produit sur Jumia)
  • Création d'une interface utilisateur avec Streamlit
  • Combiner Websraping et ML

Projets et Activités

Challenges Kaggle et Zindi

Formation sur les challenges Kaggle et Zindi pour préparer les étudiants aux compétitions de data science les plus prestigieuses.

Mini Challenges Trimestriels

Mini challenges exceptionnels (à faire en groupe) chaque fin de trimestre dans chaque niveau, avec possibilité de parrainage par certains professeurs de l'école.

Conférences et Collaborations

Conférences avec des thèmes d'actualités, partages d'expérience et collaborations avec les autres écoles du CAPESA, particulièrement pour les webinars.

Application de Suivi Académique

Réalisation d'une application pour suivi de notes et de niveau des élèves, offrant un outil pratique tout en développant les compétences des membres.

Formations

Des formations de qualités pour les membres du club afin de leur donner les bases pour être un bon data scientist

Mise en place d'une équipe de recherche

Un ensemble de personnes sera selectionné pour reflechir et travailler sur des projets pertinents.
Ce sera une equipe spécialement pour les projets du club

Compétitions opposant les différentes classes de l'école
