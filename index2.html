Notre Bureau

Pourquoi Nexus ?

"Nexus" est un mot latin qui signifie "lien", "connexion", "point de rencontre".

Nous avons choisi ce nom car il symbolise parfaitement notre ambition : faire du club un carrefour entre les esprits curieux et le monde de la data. C'est aussi l'endroit où les talents se rencontrent pour ensemble aller plus loin.

Notre Vision

Faire du club un espace collaboratif et un environnement où chaque personne peut trouver sa part de connaissances.

Nos Objectifs

  • Rendre la data science accessible à travers les formations
  • Déclencher une motivation particulière des élèves pour la data à travers les challenges
  • Favoriser la collaboration entre les membres

Module 1 : Introduction à Python et à l'Algorithmie

  • Introduction aux bibliothèques Pandas et NumPy
  • Nettoyage et transformation des données

Module 3 : Gestion et Stockage des Données

  • Classes et objets en Python
  • Organisation du code pour des projets data
Module 5 : Introduction au Machine Learning
  • Supervised vs Unsupervised Learning
  • Algorithmes de base : Régression linéaire, k-means, KNN
